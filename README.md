# Setup

### How to install?
```
wget https://raw.githubusercontent.com/harisheoran/my-personal-assistant/main/install.sh 
chmod +x install.sh
./install.sh
```

### How to stop the service?
```
systemctl --user stop my-personal-assistant.service
```
